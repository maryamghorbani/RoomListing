# Unravel app

## Super Quick

I aim to live project before you review, the project, so you can check it out here: [Unravel Live](https://livemys.com).


### Steps to proceed on local machine:
1. Run `docker-common`. # only first time
    ```bash
    cd docker-common
    docker compose up -d 
    cd ..
    ```
1. Start 
   ```bash
   make start
   ```
1. Test
   ```bash
   make test
   ```
1. E2E 
   ```bash
   make test-e2e
   ```
